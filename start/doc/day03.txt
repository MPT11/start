# day03

연산자 : 데이터를 가공하기 위해서 사용하는 기호

용어 
   ㄴ연산자		-	가공방식을 나타내는 기호
   ㄴ피연산자	-	가공에 사용되는 데이터
   
 연산자를 사용하여 가공된 결과는
 큰타입의 형태로 나온다 ex) int + double = double
 모든 연산자는 같은 연산자가 중복된 경우 왼쪽에서 오른쪽으로 진행함
 
 
 연산자 종류
 1 사칙연산자
 
 	ㄴ 기호 ( + - * / %)
 	
 	ㄴ사칙연산자는 반드시 피연산자가 2개 필요하다
 	
	ㄴ사칙연산자가 여러개 동시에 기술된 경우엔 왼쪽부터 차례로 계산
	
 2 대입연산자
 
	ㄴ 변수에 대이터를 기억하도록 하는 연산자
	
	ㄴ 기호 (=, +=, -=, *=, /=, %=)
	
	ㄴ 형식 -> 변수 = 데이터;
	
	ㄴ 반드시 왼쪽에 변수 오른쪽에 데이터가 있어야한다
	
	ㄴ 참고 ]
		변수 = 변수 + 데이터;
		==>
		변수 += 데이터;
	
	
 3 증감 연산자
 
	ㄴ 형식 ]
		++변수
		변수++
		
	ㄴ 기호 ]
		++ : 1증가시킴
		-- : 1감소시킴
		
	ㄴ 참고 ]
		모든 연산은 일반적으로 중앙처리장치에서 처리되는 것이 원칙이다
		
 4 부호반전연산자
 	기호 ]
 		

 5 비교연산자
 	ㄴ 기호 ]
 		== 같다
 		!= 같지 않다. 다르다
 		>  크다
 		<  작다
 		>= 크거나 같다 : =>는 안된다
 		<= 작거나 같다
 
 	ㄴ 주의 ]
 		사용되는 데이터와 관계없이 결과는 항상 논리값(false,true)로 나온다
 
 	ㄴ 참고]
 		이 연산자는 주로 if, for, while, ... 제어 명령에서 질문이 필요할 때
 		많이 사용되는 연산자이다.
 		
 		문자열을 비교할 경우에는 equals()를 사용해서 비교해야
 		정확하게 데이터를 비교하게된다.
 		
 		비교 연산자는 3항을 동시에 비교하진 못한다
